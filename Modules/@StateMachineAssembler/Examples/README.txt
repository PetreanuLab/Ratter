This directory contains examples of using the StateMachineAssembler.

example0.m   illustrates basic use of the StateMachineAssembler by looping through 3 states indefinitely

example1.m   illustrates basic use of the StateMachineAssembler and using 
             scheduled waves. 

example2.m   using the disassembler to get information back from the state
             machine

example3.m   using the 'no_dead_time_technology'
